This page uses the built SDK artifact from dist/ and talks to a real local backend. Use it to demo validate, enroll, and verify flows from a simple mock frontend.

Demo configuration

The demo defaults to http://localhost:8080. Keep the backend CORS setting compatible with the demo origin, or set ALLOWED_ORIGINS=* for local use.

Local demo checklist

  1. Build the SDK at the repo root before starting the demo.
  2. Run the backend locally with valid GCP credentials and the TFLite model.
  3. Use this page on localhost so camera capture works reliably in the browser.
  4. Keep internet access available for the MediaPipe detector assets used by the SDK.
