Demo checklist

  1. Build the SDK at the repo root before starting the demo.
  2. Use the hosted backend by default, or run your own backend and override the API URL in the form.
  3. Use this page on localhost so camera capture works reliably in the browser.
  4. Keep internet access available for the MediaPipe detector assets used by the SDK.
