SimFace SDK Demo App

This page uses the built SDK artifact from dist/ and starts with a local backend URL by default. Override the API URL below if you want to point the demo at a different backend.

Demo configuration

The demo defaults to http://localhost:8080 so public clones start against a local backend. Override the API URL at any time. If you point the demo at a different backend, keep its CORS setting compatible with the demo origin.

For safety, the demo only stores the API URL, presentation mode, and client ID in local storage. Project IDs and API keys are kept in memory for the current page only, so prefer short-lived credentials during testing.

Inline SDK capture

The demo passes this `simface-capture` element into the SDK helpers for embedded capture instead of opening a separate popup.
